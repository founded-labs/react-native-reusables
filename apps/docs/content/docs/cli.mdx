## init

Use `init` to scaffold a new React Native project using our starter template.

<CommandTabs args={["init"]} />

### Options

```bash
Usage: react-native-reusables init [options]

Initialize a new React Native project with reusables

Options:
  -c, --cwd <cwd>    the working directory. defaults to the current directory.
  -h, --help         display help for command
```

## add


Use `add` to add React Native components to your project.

<CommandTabs args={["add", "[components]"]} />


### Options

```bash
Usage: react-native-reusables add [options] [...components]

Add React Native components to your project

Arguments:
  components         name of components to add

Options:
  -c, --cwd <cwd>        the working directory. defaults to the current directory.
  -y, --yes              skip confirmation prompt
  -o, --overwrite        force overwrite of existing files
  -a, --all              add all components
  --path <path>          path to the component
  -h, --help             display help for command
```

<Callout type="info">
  Uses the [shadcn](https://ui.shadcn.com/docs/cli#add) CLI under the hood.
</Callout>


## doctor

Use `doctor` to check if everything is set up correctly.

<CommandTabs args={["doctor"]} />

### Options

```bash
Usage: react-native-reusables doctor [options]

Check your project setup and diagnose issues

Options:
  -c, --cwd <cwd>        the working directory. defaults to the current directory.
  --summary              show summary only
  -y, --yes              skip confirmation prompt
  -h, --help             display help for command
```
